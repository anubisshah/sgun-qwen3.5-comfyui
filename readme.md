# ComfyUI-Qwen3.5-Sgun 🚀

这是为 ComfyUI 深度适配的 **Qwen3.5** 本地化节点插件。支持通义千问最新发布的 Qwen3.5-4B 与 Qwen3.5-9B 全系列模型，涵盖了高性能文本生成与多模态视觉理解（VL）能力。

---

## ✨ 核心特性

- **最新模型适配**：完美支持 Qwen3.5-4B (轻量速度王) 与 Qwen3.5-9B (性能甜点位)。
- **全能视觉支持**：具备多模态处理能力，支持连接 ComfyUI 图像输出进行视觉问答或图像描述。
- **智能路径管理**：
    - 自动扫描 `ComfyUI/models/LLM` 文件夹。
    - 支持通过文件夹名称或绝对路径加载本地模型。
    - 支持缺失模型自动从 Hugging Face / 魔搭社区下载。
- **精度切换**：支持 `bf16` (推荐)、`fp16` 与 `fp32` 精度，适配不同世代的显卡。
